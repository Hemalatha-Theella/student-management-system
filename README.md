# Student Management System (Java)

## Description
A console-based Java application to manage student records.

## Features
- Add student
- View all students
- Search student by ID
- Delete student

## Technologies Used
- Java
- ArrayList
- OOP Concepts

## How to Run
1. Compile:
   javac Student.java StudentManagementSystem.java
2. Run:
3.  java StudentManagementSystem
